=========================================================
   УСТАНОВКА
=========================================================

1. Запустите файл SubscribersRegistry_Setup.exe
2. Выберите язык установки
3. Прочитайте лицензионное соглашение
4. Выберите папку для установки (по умолчанию: Program Files\Ростелеком)
5. Выберите дополнительные задачи (ярлыки, автозагрузка)
6. Нажмите "Установить"
7. Дождитесь завершения установки
8. Запустите программу
